# MRapor Admin Panel

Admin panel uygulaması. React, TypeScript, Tailwind CSS ve shadcn/ui kullanır.

## Kurulum

```bash
# Bağımlılıkları yükle (YARN kullan!)
yarn install

# .env dosyası oluştur
cp .env.example .env

# Development modunda çalıştır
yarn dev

# Build
yarn build

# Preview build
yarn preview
```

## Özellikler

- Kullanıcı yönetimi (CRUD)
- Kullanıcı hizmet bitiş tarihleri
- Kullanım istatistikleri
- Client uygulamasına admin olarak bağlanma
- SQL sorgu yönetimi
- Rapor tasarlama

## Teknolojiler

- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui
- Vite
- React Router
- Axios

## Port

Development: `http://localhost:3001`
